# EJ-RG-2
Este repositorio fue creado simplemente para practicar y entender cómo funciona la sintaxis de Markdown en los archivos README. No es un proyecto serio, solo una guía y espacio de pruebas para acostumbrarme a documentar proyectos en GitHub.

## Instalación y ejecución
1. Creamos un nuevo repositorio desde la web de GitHiub.
2. Configuramos la privacidad y los permisos.
3. Se abre el archivo README.md y lo editamos con titulos, listas, enlances y imagenes.
4. guardamos cambios y visualizamos en GitHub.

 ## Características
 
- Ejemplos básicos de README
- Explicaciones claras y organizadas 
- Incluye comandos y fragmento de código
